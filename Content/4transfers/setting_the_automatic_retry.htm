Automatically retrying a failed transfer

If the initial attempt to transfer a file fails, WS_FTP can automatically retry the attempt a set number of times.

To set the Automatic retry:

  1. Click the Options button on the toolbar, or Options from the Tools menu.

  2. Select the Transfers page.

  3. In the Number of retries for failed transfers box, enter the number of times you want WS_FTP to retry the transfer. (0-99)
