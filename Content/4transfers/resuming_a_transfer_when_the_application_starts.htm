Resuming a transfer when the application starts

WS_FTP can automatically resume interrupted transfers or transfer attempts when the application is started, if WS_FTP is terminated while: 

It is in the process of transferring a file 

- or-

If it is still on the Transfer list as a failed transfer.

To set this option:

  1. In the toolbar, click Options or select Options from the Tools menu.

  2. Select the Transfers page.

  3. Select the Automatically resume transfers when application starts option.

  4. Click OK.
