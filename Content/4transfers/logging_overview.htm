Logging Overview

Log files are a record of the conversation between the client (WS_FTP) and the FTP server you are connected to. There are three ways you can view logs generated by WS_FTP.

  • The Connection Log window displays all conversations between WS_FTP and the server. The Connection Log window is displayed in the Information WIndow at the bottom of the application. Click Connection Log to display the log.

  • In the toolbar, click Options, then select the Logging dialog. Select the log file from the directory identified.

  • In the Transfer Manager, select the transfer item, right-click and select Log.
