Transferring files or folders

To transfer files or folders:

  1. Open the folder to which you want to transfer files on the destination system.

  2. Select the file(s) or folder(s) on the source system.

  3. Do one of the following:

    • Drag the selected files to the destination folder or the destination connection tab.
    • Click the transfer arrow to transfer the selected file(s) or folders(s) to the destination system.
    • Select Edit > Copy, then select the destination connection tab and select Edit > Paste. You can also use keyboard shortcuts and right-click menus to copy and paste.  

To get prompted for a file name:

Do one of the following:

  • Click a transfer arrow when no file is selected.

  • Hold the Ctrl key while clicking the transfer arrow (When a file is selected).
