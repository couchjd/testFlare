Specifying the Startup Transfer Mode

The Startup Transfer Mode is the transfer mode that will be used unless you override before you make the transfer.

To set the Startup Transfer Mode:

  1. From the Site Manager, select the site in the Sites list.

  2. Click the Edit button.

  3. Select the Startup dialog.

  4. Under Startup transfer mode, select one of the following:

  • ASCII. Used for text file transferring.

  • Binary . Binary mode is used for transferring files that are not text (e.g., executable programs, word processing documents, spreadsheets, databases, graphics files, and sound files).

  • Auto Detect. When selected, all files are transferred in binary mode unless the extension is listed on the Extensions tab. Extensions can be up to ten characters long and can contain periods. (These are not DOS extensions!) If you expect it to match on a period, you must enter the period.
