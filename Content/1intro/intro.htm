Introduction

Where do you want your files to go today? Check out our online resources to learn how to make the best use of Ipswitch WS_FTP.

Getting Started Tour (Flash player required) New to WS_FTP? Check out this video tour to view the major features in action. From start to finish, the tour takes about 12 minutes, or you can choose to view segments of the tour.   

Getting Started Guide (PDF - 547Kb) This 4-page quick start takes you through the initial installation and connection to an FTP server.

Tools Guide (PDF- 336 Kb) Covers advanced features such as Batch Mode, using the Transfer Manager, and the Find, Synchronize, Scheduler, Script, and command line utilities.

Security Guide (PDF- 219 Kb) Covers the security protocols that work with WS_FTP, including SSL, SSH, and PGP, and how to work with firewalls.
