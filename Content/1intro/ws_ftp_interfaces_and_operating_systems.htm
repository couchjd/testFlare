WS_FTP Interfaces and Supported Operating Systems

WS_FTP provides an interface that displays file lists in two panes. By default, the left pane is a view of the local system, and the right pane is a view of a remote system. The interface provides tabs above the file list panes that allow management of multiple local and remote views..

WS_FTP supports the following Operating Systems:

  • Windows 98
  • Windows NT
  • Windows 2000
  • Windows ME
  • Windows XP

Other FTP Products

The WS_FTP Server and the WS_FTP SDK must be purchased separately.
