Getting prompted for path and file name

When you transfer a file, WS_FTP can automatically display a prompt that shows you the proposed destination path and file name and allows you to confirm or change them.

To get a prompt for transfers:

  1. From the Site Manager, select the site in the Sites list.

  2. Click the Edit button.

  3. Select the Transfer page.

  4. Select Prompt for Destination File Names.
