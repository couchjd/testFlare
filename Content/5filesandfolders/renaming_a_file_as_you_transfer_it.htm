Renaming a file as you transfer it

You can get a prompt that allows you to enter a path and name for each file you transfer to/from the site. You can also rename just one file as you transfer it as follows:

  1. Select the file.

  2. Press and hold the Ctrl key.

  3. Click the Upload or Download arrow. You are prompted for a file name and location.

  4. Specify the location (path) and enter a new file name.

  5. Click OK.

If the name is not accepted, you may be trying to use a long name on an FTP site that does not allow long names. Try a name of eight characters or less.
