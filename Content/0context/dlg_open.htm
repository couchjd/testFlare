Open (dialog box)

If you select Open it and the source location is an FTP site, WS_FTP will transfer the file to the Temp folder in your Windows directory and then execute the file according to its file extension.

If you select Save it to disk, you will be prompted for a path and file name.

Select the transfer mode. Files can be transferred in ASCII, binary, or "Auto" mode.

A file that appears to be text, postscript, or other ASCII format should be transferred in ASCII mode. Note: Transferring a binary file in ASCII format can damage the file.

A file that appears to be data, an executable, or compressed should be transferred in binary mode. (Binary mode should also be used if there is no file extension or the file extension is not registered.)

If you select Auto mode, WS_FTP checks to see if the file has an extension that is associated with a text viewer. If so, ASCII mode is used; if not, binary mode is used.
