Download Dialog

You have chosen to download a file from an FTP site using WS_FTP. You can either open this file or save it to disk.

  • If you choose to Open the file from its current location, WS_FTP will copy the file to your Temp folder, and then open the file in its associated application.

  • If you choose to Save it to disk, you will be prompted for a path and file name.

  • If you choose to Reschedule, the download will be cancelled and the WS_FTP Scheduler Utility appears.
