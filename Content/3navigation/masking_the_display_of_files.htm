Masking the display of files and folders

If your PC or an FTP site contains a large number of folders and files, you can limit the items displayed by using the File Mask text box (in the command bar). Note: The File Mask is not enabled on the command bar by default.

You can mask the display in two ways: set it for a particular site, or set it "on the fly."

 

Setting the file mask for future connections to a particular site:

  1. Click Connect to open the Site Manager.

  2. Select the site in the Sites list, then click Edit.

  3. Select the Startup page.

  4. To limit the displayed files, enter characters and an asterisk in the Local file mask and/or Remote file mask text box. Examples: To display only text files, enter *.txt. To display only files whose names begin with the letter A, enter a*. If you are connected to a UNIX FTP site, you can use:

-al or -la to view hidden (.) files

 

Setting the file mask "on the fly" for just the current connection:

You can use a file mask while navigating a particular site. There is a File Mask box in the command bar: one for the local system (your PC) and one for the remote site (FTP site). In the appropriate File Mask text box, enter a file mask and then press Enter.
