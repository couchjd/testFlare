Changing the current folder

Do one of the following:

  • If the desired folder name is shown in the list box, click it.

  • Select the desired folder and click the Change Folder button. WS_FTP changes the current folder to the one selected.

  • To get a prompt where you can enter the desired folder name, make sure nothing is selected in the list box and click the Change Folder button. When prompted for a folder, you can also enter a drive letter followed by a colon (for example, d:) to change to a different disk drive.

  • Edit the folder path in the current folder box.

  • Click the Browse button next to the folder and select the folder from the list.

Note: If you selected Save folder history when connection is closed on the Display tab of Program Options, the drop-down list for both the Local System and Remote Site will show the folders you visited during previous sessions.
