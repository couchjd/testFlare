Sorting files and folders

You can sort the files and folders in the Local System or Remote Site list boxes by using the column headings (Name, Date, Size) above each list or the Arrange Icons command on the right-mouse menu. You can sort all views by selecting View from the main menu and then selecting the Arrange Icons By item.  

Using column headings to sort:

Click a column heading to sort the files and folders as follows:

Column Description
Name

Click to toggle the alphabetical list between ascending and descending order.

Click Ctrl+Name to sort the file names by file extension.
Date Click to toggle the list between ascending and descending order in terms of their file date.
Size Click to toggle the list between ascending and descending order in terms of their file size.
Type Click to group the items by type.
