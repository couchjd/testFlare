Importing an SSH Key Pair

You can import an SSH key pair for use in authenticating an SSH connection to a server. You must have both the public and the private keys to import. 

To import an SSH key pair:

  1. From the main window, select Tools > Options. The Program Options dialog appears. 

  2. Select SSH > Client Keys.

  3. Click Import. The SSH Client Key Import wizard appears. 

  4. Follow the on-screen prompts to complete the wizard.
