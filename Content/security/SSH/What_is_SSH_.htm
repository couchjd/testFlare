What is SSH?

SSH (Secure Shell) is a security application that allows you to make a secure connection to a server that has the SSH protocol installed. Whereas FTP servers usually 'listen' on port 21 for connection, SSH servers use port 22. 

For maximum security, Ipswitch WS_FTP Pro utilizes SSH2 with the SFTP protocol to make SSH transfers.

Note: SSH can be used with a variety of authentication methods. However, WS_FTP only supports simple password authentication and public key authentication.
