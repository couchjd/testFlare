How to make an SSL connection

To make an SSL connection with a server you have an account on:

  1. Follow the directions for configuring a site, being sure to enable SSL by selecting FTP/SSL or FTP/Implicit SSL Server type.

  2. Once you click Connect, WS_FTP 'tells' the server that you want to make an SSL connection. The server then transmits to you an identifying certificate, letting the client know who the server is. If that certificate is already listed in your Trusted Authority database, the connection will be made.

  3. If that certificate is not listed as a trusted authority, the Non-Trusted Authority window will appear.

  4. Select the option you need and click OK. If the server does not require a certificate to be returned, the secure connection will be established. All data transmitted between you and the server will be encrypted.

If you are attempting to make a connection to a server which asks WS_FTP to send back a certificate, follow the directions for Client Certificate Verification.
