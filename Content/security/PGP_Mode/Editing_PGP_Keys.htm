Editing PGP Keys

After keys are created or imported, you can use the PGP Key Edit dialog to digitally sign them, or, if they are already signed, to revoke the signature. You may also revoke the entire key, which will prevent it from being used in the future. 

To sign an key:

  1. From the main window, select Tools > Options. The Program Options dialog appears. 

  2. Select PGP > Keys.

  3. Select the key you wish to sign, then click Edit. The PGP Key Edit dialog appears. 

  4. Click Sign. The PGP Sign Key dialog appears. 

  5. Select Expiration. Never ensures the signature will never expire, while Expires on allows you to specify a date after which the signature will not be honored. 

  6. If you would like your signature to be attached to the key when it is exported, select Exportable. To export the key without your signature, clear Exportable. 

  7. Click OK. The Signing Key and Passphrase dialog appears. 

  8. Select a Signing Key and enter its accompanying  Passphrase.

  9. Click OK. The key is now signed. 

To revoke a signature on a previously signed key:

  1. From the main window, select Tools > Options. The Program Options dialog appears. 

  2. Select PGP > Keys.

  3. Select the key from which you wish to revoke a signature, then click Edit. The PGP Key Edit dialog appears. 

  4. Select the Certificate tab.

  5. Expand the key tree to locate the signature you wish to revoke. Select it, then click Revoke.   

  6. A prompt appears. Click OK. 

  7. Enter the passphrase for the signature you wish to revoke. Click OK to complete the process. 

Note: Once the signature is revoked, you can also delete it from the key by selecting the revoked signature and clicking Delete.

 

To revoke an entire PGP key:

  1. From the main window, select Tools > Options. The Program Options dialog appears. 

  2. Select PGP > Keys.

  3. Select the key from which you wish to revoke a signature, then click Edit. The PGP Key Edit dialog appears. 

  4. Click Revoke.  

  5. A prompt will appear. Click  Yes.

  6. Enter the passphrase for the key you are revoking.

  7. Click OK to complete the process.

Note:  Revoking an entire key cannot be undone. Once a key is revoked, it cannot be reinstated.
