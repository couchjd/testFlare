Importing PGP Keys

In order to encrypt files for others to open using PGP Mode, you will need to import their PGP keys. You can also import keys and keyrings that you use in another PGP application.  

Note:  WS_FTP can only access keys and keyrings that exist as standalone files. If the keys or keyrings you wish to import are embedded in an e-mail or in a PGP application, you must export them prior to importing them with WS_FTP. 

To import a PGP Key or Keyring:

  1. From the main window, select Tools > Options. The Program Options dialog appears. 

  2. Select PGP > Keys.

  3. Click Import. The PGP Key Import Wizard appears.   

  4. Follow the on-screen directions to complete the import process.
