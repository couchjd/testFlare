Generating a PGP Key Pair

If you do not already have a personal PGP Key Pair, WS_FTP can help you generate one.

To generate a PGP Key Pair:

  1. From the main window, select Tools > Options. The Program Options dialog appears.

  2. Select PGP > Keys.

  3. Click Create. The PGP Key Generation Wizard appears. 

  4. Follow the on-screen directions to complete the key creation process.
