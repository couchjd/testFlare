MDTM Error Message

This message means that the current FTP site does not support Month-Day-Time-Marking; this means that after the file was uploaded, WS_FTP attempted to preserve the date and time, but the time cannot be preserved by this FTP site. However, files will transfer correctly, and there will be an updated timestamp from the FTP site on the files.

Note that this is not a fatal error, even though it displays in red.

You can disable this option by selecting Preserve filetime on upload (MDTM support required) on Site Properties: Transfer.
