Using Active Edit

The active edit feature allows you to edit remote files quickly and without having to actively transfer them before and after editing.

Once active edit has been configured for an extension, you can launch the file on the remote system.

To use active edit on a remote file:

  1. Configure an editor and extension.

  2. Select the file you want to edit.

  3. Click Edit from the command bar, or right-mouse click on the file and select Operations >Edit.

  4. The file launches in the associated application.

To display information about active edits:

  • Select the Active Edit Manager button on the toolbar or select View from the main menu and then select Active Edit Manager. (This option is only available if the application is currently editing remote files). 

To edit a local file: 

  1. Select the file you want to edit.

  2. Click Edit from the command bar, or right-mouse click on the file and select Open.

  3. The file launches in the associated application.
