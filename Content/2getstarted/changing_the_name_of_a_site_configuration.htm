Changing the name of a site configuration

The Site Name is a name you assign to an FTP site for your own convenience; this is the name you will select from the list of configured sites; it can be anything you like, but if you enter more than 78 characters, the name will be truncated. Examples:My Web Site, Department FTP Server, and Client DropBox.

  1. In the toolbar, click Connect to open the Site Manager.

  2. Click Rename or right-click the site and select Rename from the right-mouse menu.

  3. Enter a new Site Name.
