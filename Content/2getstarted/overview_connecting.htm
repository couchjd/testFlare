Overview: Connecting to FTP Sites

Before you transfer files to or from an FTP site, you must first connect to the site. You can connect in one of several ways:

  • Selecting a configured site

  • Selecting a site you visited recently

  • Configuring an FTP site you intend to re-visit

  • Using the Site Manager

  • Using the Address bar 

  • Using the Connect Menu

  • Automatic Reconnect
