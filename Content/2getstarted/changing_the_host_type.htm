Changing the host type

The host type is usually the manufacturer and model of the FTP server(The physical device on which the FTP site resides). For a complete list of host types supported by WS_FTP, go to the KnowledgeBase on the Ipswitch Web site and search for "host type."

  1. In the toolbar, click Connect to open the Site Manager.

  2. Select the site in the Sites list.

  3. Click Edit.

  4. Select the Advanced dialog.

  5. Select the Host Type: 

  • If you know the host type, select it from the list.

  • If you do not know the host type, WS_FTP can probably detect it for you: try "automatic detect."

If you would like to figure out the host type, use the procedure in the topic, File list is missing, blank, or unreadable.
