Automatic Reconnect

You can control what happens with WS_FTP each time you start the application. With the Automatic Reconnect function, you can choose to either:

  • Connect to a default site
  • Connect to the last site you connected to
  • Display the Site Manager
  • Display a blank workspace

This feature is configured on the General page of Program Options.

This feature is particularly useful if you have only one or two sites that you connect to on a regular basis. By bypassing the connection process, you can get to your transfers after only a couple of clicks.
