Selecting a Default Site

With WS_FTP, you can select a site from your list and make it your default site. You can then use the automatic reconnect feature to have WS_FTP connect to that site when you start the application.

To select a default site

  1. In the toolbar, click Connect to open the Site Manager.

  2. Right-click on the site name.

  3. Select Default Site from the pop-up menu.
