Transfer Window

The Transfer Manager window displays a list of all pending uploads and downloads, as well as active transfers. This list can be sorted by clicking on the column headers.

You can use the Transfer Window to verify transfers, pause or cancel a transfer, troubleshoot both transfer and connection problems, or interact with "transfer jobs" shown in the Transfer Manager and Transfer History. For example, you can use Batch Mode with the Transfer Manager to build a list of files for transfer.

By default, the Transfer Window is viewable, but you toggle it on or off using the View menu.
