Conversion Rule Explanations

The following icons are used to indicate whether file extensions are changed during uploads and downloads.

 

  - File extensions are changed only as files are downloaded.

 -  File extensions are changed only as files are uploaded.

 -  File extensions are changed as files are both uploaded and downloaded.

 

Example

.xyz .txt means that if file.xyz is uploaded, it will be changed to file.txt. However, file.txt will remain file.txt if it is downloaded.
